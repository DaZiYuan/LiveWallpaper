## 开源动态壁纸软件，免费版 wallpaper engine

### 版本特性：

- 壁纸核心端
  - [x] 提供 API 驱动壁纸核心功能
  - [x] 开机启动核心端即可运行壁纸
  - [x] 弱 UI 减少资源占用，仅提供托盘
  - [ ] 增加exe壁纸支持
- UI 端
  - [ ] 社区分享，此功能 UI 端提供
    - [ ] 壁纸投稿
  - [ ] Nuxt 实现，暂未开源。（尝试了Flutter，Winui，最终决定还是传统web效果最好，最不占资源）

### 编译环境：

- VS2019
- Win10

### 分支管理
- master 是最新版代码（当前是2.0)，开发中不完全可用
- 1.x 目前线上WPF开发的版本
